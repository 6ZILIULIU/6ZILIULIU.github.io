vue3.x从安装到部署

Sep 3, 2019

首先需要安装node和npm

  • 具体安装方法在菜鸟教程或官方网站有。

安装vue-cli3

1.全局安装淘宝镜像cnpm

  • npm install -g cnpm
  1. 全局安装vue脚手架
  • cnpm install -g vue-cli
  • cnpm install -g vue
  • 安装完可能会出现版本太低(vue create myproject 命令只能vue-cli3才能使用，2.x用的是vue init webpack myproject)

  1. 创建一个vue项目
  • vue create myproject
  1. 进入项目文件夹，运行
  • cd myproject
  • npm run serve
  1. 打包部署到nginx上
  • npm run build
